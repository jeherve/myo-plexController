## Plex Controller

Control Plex Theatre without leaving your couch, thanks to Myo.

**This is a work in progress!** Pull requests are always welcome!

## Controls

- Wave in:  Left
- Wave out: Right
- Fist: Play / Pause
- Spread fingers: Escape
